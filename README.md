# Student Material Design

## Цель работы

Научиться применять принципы Material Design в Android приложениях с использованием Jetpack Compose. В результате лабораторной работы
будет создано приложение «Список студентов», которое демонстрирует использование цветовых схем, типографики, форм и анимаций Material Design.

## Функциональность

* Отображение списка студентов с фотографиями
* Кастомная цветовая схема (основной цвет — зелёный)
* Поддержка светлой и тёмной темы
* Круглые аватарки студентов
* Карточки с уникальными скруглениями углов
* Кастомные шрифты
* Верхняя панель приложения с логотипом
* Раскрывающиеся карточки с описанием студентов
* Плавная анимация при раскрытии/сворачивании

## Технологии

* Язык программирования Kotlin 
* Jetpack Compose
* Material Design 3
* Шрифты: Montserrat, Abril Fatface
* Иконки: Material Icons Extended
* истема контроля версий: Git/GitHub
